## Getting Started
### Installation
Pull it in your project folder
```commandline
git clone [YOUR REPO URL] [YOUR PROJECT NAME]
cd [YOUR PROJECT NAME]
```

## Running the application
Start both frontend and backend with live reloading:
```bash
docker compose up --build --watch
```

## Stopping the application
```bash
docker compose down
```# SettleOps-AI
